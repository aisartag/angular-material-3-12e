Che cos'è NgRx?

NgRx è un framework per la creazione di applicazioni reattive in Angular.

E' costituito da una serie di pacchetti con scopi specifici. Per saperne di più vai al sito del team NgRx.

Angular nelle ultime versioni (a partire dalla 16 - siamo attualmente alla 18) ha rilasciato una nuova funzionalità: signals.

Consulta Angular University - Signals.

Il team NgRx ha prodotto il pacchetto @ngrx/signals per una gestione professionale dei segnali.
