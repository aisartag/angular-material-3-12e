NgRx Signals è una libreria autonoma che fornisce una soluzione di gestione dello stato reattivo e un set di utilità per Angular Signals.

Caratteristiche principali

  • SignalStore: una soluzione di gestione dello stato completa che fornisce supporto nativo per Angular Signals e offre un modo efficace per gestire lo stato dell'applicazione.
  • SignalState: un'utilità leggera per gestire lo stato basato sul segnale nei componenti e nei servizi Angular in modo conciso e minimalista.
  • Integrazione RxJS: un plugin per l'integrazione opt-in con RxJS, che consente una gestione più semplice degli effetti collaterali asincroni.
  • Entity Management: un plugin per manipolare e interrogare raccolte di entità in modo semplice ed efficiente.
